  * Pseudocode:
```cpp
//T(n) = theta(n^2)
Find-Max-Profit-Brutally(prices)
  def max = tuple(buy = 0, sell = 1)
  if prices.length > 1
    for b = 0 to prices.lengh - 1
      for s = b + 1 to prices.lengh - 1
        if prices[s] - prices[b] > prices[max.sell] - prices[max.buy]
          max = tuple(b, s)
  return max
```
